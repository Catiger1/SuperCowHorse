* 游戏介绍
>* 游戏属于局域网对战游戏，玩家发射子弹并击退敌人造成坠落或者碰撞陷阱得分，每次对局开始前玩家都可以选择放置一次陷阱
* 游戏操作
>* 通过AD或者方向键控制人物移动，通过J键发射子弹
